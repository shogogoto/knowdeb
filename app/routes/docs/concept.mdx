# コンセプト

**knowde**は知識(know)を結び目(node)に解きほぐして依存関係を整理するための**メモツール**です。
それを(単文)同士の関連付けとして独自文法で表現します。

## 用語
- 単文, sentence: 1行の短い文。知識ネットワークを成すノード
- 用語, term: 単文に付けられた名前
- 定義: 単文に用語を与えること。または単語と用語の組
- リソース: 単文が属するネットワーク単位。メモのファイルに相当。
- フォルダ: リソースやフォルダの入れ物。
- エントリー: リソースとフォルダの総称
- 引用:
## 単文の関連付けタイプ
単文の関連づけには4種類あります。
- part: 詳細を補足する、されるの関係
- logic: 前提と結論を表す論理的な関係
- ref: 用語を引用してより高度で複雑な用語を定義する
- meta:
単文の親子関係

単文には名前を付けられます(用語)。

## メモする過程が大事
理解は分解と再構成の2ステップによって成されます。
例えば、風景を見て把握するためのステップは以下です。
1. 分解する
  - 目立つもの、支配的なものを注視する
  - その他は目立つものの周りに従属的に配置されているように見える
2. 再構成する
  - 最も目立つものとその直近の次に目立つものの位置関係を把握し直す
3. これを要素がなくなるまで繰り返す
4. 精神に全ての要素が同時に現れるようになる
