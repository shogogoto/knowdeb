# docs/overview ネットワーク図 サーバーとクライアントの関係

クライアント PC/スマホ で ファイル作成
手元のファイルはGitでも管理できる
ブラウザやCLI からサーバーへ送信・同期
サーバーはグラフデータベースのneo4j Auraと通信

```mermaid
graph TD
    subgraph クライアント
        A[クライアント PC/スマホ] --> B(ファイル作成);
        B --> C{手元のファイル};
    end

    C -- Git管理 --> D[Gitリポジトリ];

    subgraph サーバー連携
        A -- ブラウザ/CLIで送信・同期 --> E[サーバー];
    end

    E -- 通信 --> F[Neo4j Aura];

```
